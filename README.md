# LtMotorLib电机控制库
LtMotorLib电机控制库是一套电机控制的软件解决方案，支持BLDC和PMSM。采取分层架构设计：应用层代码实现通讯与保护等功能；封装PID、六步换向、FOC等常用算法模块；通过一套精简的设备框架，既隔离了底层硬件驱动，又保证了电机控制算法的高效执行；硬件驱动层将传统的硬件配置归纳为电流传感器、通讯模块、位置传感器和定时器等六大模块，结构清晰，拓展性好。可采用宏开关启动调试功能，在开发阶段快速定位故障。库的底层源码采用面向对象和模块化设计相结合的方式实现，调用API接口直观明了。本项目已在STM32G474平台完成验证，项目持续更新中，同步提供MATLAB/Simulink仿真，不定期发布相关技术文档。
